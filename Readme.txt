工大圈子

构建一个属于工大人自己的圈子，该圈子可以很好的共享工大的信息，并且推荐每天today(today.hit.edu.cn)网站上的热点新闻、热点人物及热点话题。

功能至少包括：热点新闻推荐，热点话题和热点人物推荐，好友关注，用户留言等。

注意：采用浏览器/服务器方式实现或手机APP/服务器方式实现，如果采用浏览器/服务器模式，要求云端配置服务器。

第一次迭代前由于爬取数据使用python，向MySQL中传送数据时出现中文的乱码问题，在python和MySQL均不兼容中文的情况下仅解决了MySQL的兼容性问题，时间开支远大于预期，因此总进度减慢。将爬取内容存入了todayhit文件夹里，以压缩包形式上传。
